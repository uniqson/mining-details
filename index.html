MINING PROCESS

Mining in UniqCoin is performed with a mathematical formula. The total coins created is 50000 per Genesis Address. As we have 10 Genesis Addresses to be mined  into per day. 

[TotalGenesisAddresses] * [RewardPerAddress] * [TotalEthBlocksPerDay] = Total coins generated per day 500k per day

  function executeDailyMining() public returns (bool) {
    require(block.number.sub(currentBlockNumber) >= minimumBlocks);

    currentBlockNumber = block.number;
    for(uint i = 0 ; i < totalRewardAddress; i++) {
      _mint(_rewardAddress[i], perAddressToken);
    }

    return true;
